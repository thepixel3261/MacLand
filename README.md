# My Arch Hyprland Dotfiles
Not supposed to represent MacOs, just the glassy style!

(Demos made in CachyOS)
![Demo](demo.png)
![Demo-dark](demo-dark.png)
To install on Arch based distro:

Install on a fresh Minimal Arch installation for best results

1.
```bash
git clone https://github.com/thepixel3261/MacLand.git ~/dotfiles
```

2.
```bash
cd ~/dotfiles
```

3.
```bash
./install.sh
```
